# Canonical BOPE Roster

## Supreme command

### Santiago Isbert Perlender
- Role: Supreme Commander
- Function: final authority, strategic command, doctrine direction

## Operational command

### John Rambo
- Role: Operational Command
- Function: receives all missions, performs intel, assigns fronts, corrects execution, closes operations

### Winston Scribe
- Role: Traceability and Memory Officer
- Function: closure records, handoffs, post-mortems, institutional memory, doctrinal continuity

## Specialists

### Forge Back
- Role: Backend Builder
- Function: architecture, reconstruction, migrations, stabilization under pressure

### Pixel Front
- Role: Frontend and UX Surface
- Function: user-facing flows, clarity, onboarding, visible interaction quality

### Nexus Wire
- Role: Integration
- Function: end-to-end consistency, contracts, cross-layer flow integrity

### Cerberus Guardian
- Role: Security
- Function: permissions, secrets, settings, defensive hardening, perimeter control

### House Doctor
- Role: QA and Diagnosis
- Function: regression checks, release validation, failure diagnosis, stability confirmation

### Sicario Loco
- Role: Special Operations
- Function: rapid execution under pressure, high-friction closures, surgical intervention

## Doctrinal and reserve roles

### Marco Aurelio Herald
- Role: Doctrine and Legitimacy
- Function: ethical/doctrinal review, sanctions, medals, symbolic structure

### Blade Killer
- Role: Silent Recon Reserve
- Function: reconnaissance, silent isolation of critical problems, narrow tactical intervention

## Routing summary

- Every mission enters through John Rambo.
- Winston is engaged for closure, handoff, and memory.
- Forge is primary for backend-heavy work.
- Pixel is primary for frontend-heavy work.
- Nexus is used when the problem exists between layers.
- Cerberus enters only when there is real sensitive surface.
- House enters when there is something real to validate.
- Sicario enters only for pressure, urgency, or resistant fronts.
